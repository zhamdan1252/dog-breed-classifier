IN ORDER TO RUN THIS SCRIPT THE DIRECTORY SHOULD BE SET UP AS FOLLOWS

- In the main directory, there should be the 'breeds.csv' file, the 'generate_breeds_key.py',
   and the 'run_training.py' script. 

- There should be three subdirectories found in the home directory: 'img', 'layers_package', and 'out'
    - img should contain the directory of images for each dog breed.
    - layers_package contains all the layer scripts for the deep learning model
    - out is the location where the final cross entropy loss plot.
